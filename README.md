# Open-Source-Project

SMART TECH: E-commerce System 

SMART TECH E-Commerce System is a web-based online shopping platform developed for selling computer hardware components and accessories. It provides customers with a convenient way to browse different products, view product details, add items to their shopping cart, place orders, submit product reviews, and track the status of their purchases. The system is designed to make the shopping process easier, faster, and more organized for users who are looking for computer-related products.

In addition, the system includes an administrator side where admins can manage products, categories, customers, orders, and other important records. Administrators can also monitor the overall performance of the business through analytics dashboards, which help present useful information such as sales trends, recent orders, product performance, and customer activity. Overall, SMART TECH E-Commerce System aims to provide a simple, efficient, and user-friendly online shopping experience while also helping administrators manage the business more effectively.


INSTALLATION GUIDE
System Requirements
•	PHP 8.x – Required for running the backend of the system.
•	MySQL 8.x – Required for database management and data storage.
•	XAMPP – Used to run Apache and MySQL locally.
•	Modern Web Browser – Used to access and test the system.
Installation Steps
Step 1 – Download or clone the project repository.
Step 2 – Place the project folder inside the following directory:
xampp/htdocs/
Step 3 – Open the XAMPP Control Panel and start Apache and MySQL.
Step 4 – Create a new database in phpMyAdmin.
Step 5 – Configure the database credentials in:
Database/db.php
Step 6 – Run the following file to create the database tables:
Database/initdb.php
Step 7 – Access the system using the browser:
http://localhost/SMART-TECH-E-Commerce-System-Project-main
